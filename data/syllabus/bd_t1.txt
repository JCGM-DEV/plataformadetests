¿Cuál de los siguientes formatos de archivo representa la información en una estructura tabular fija de filas y columnas?
JSON
XML
*CSV
Fichero de texto plano sin formato
EXPL:CSV (Comma-Separated Values) organiza los datos en filas y columnas con un separador fijo. JSON y XML son semiestructurados y jerárquicos. Un fichero de texto plano no tiene estructura definida.
En el modelo Entidad-Relación, el conjunto de objetos del mundo real que comparten los mismos atributos y se identifican con un nombre común se denomina?
Relación
Atributo
Tupla
*Entidad
EXPL:Una entidad representa un objeto del mundo real con existencia propia (CLIENTE, PRODUCTO). Una relación es la asociación entre entidades. Un atributo es una propiedad de la entidad. Una tupla es una fila de una tabla relacional.
Según la clasificación de bases de datos basada en su modelo lógico, ¿cuál de las siguientes opciones NO pertenece a dicha clasificación?
Jerárquicas
Relacionales
Orientadas a objetos
*Empresariales
EXPL:Las clasificaciones por modelo lógico son: jerárquicas, en red, relacionales y orientadas a objetos. "Empresariales" no es una categoría del modelo lógico sino un ámbito de uso o sector de aplicación.
¿Qué tipos de fichero permiten incorporar medidas de seguridad como contraseñas y encriptación?
Planos
Secuenciales
Multimedia
*Indexados
EXPL:Los ficheros indexados permiten incorporar medidas de seguridad como contraseñas y encriptación, además de acceso directo a registros. Los ficheros planos son simples secuencias de texto sin estas capacidades.
¿Entre qué tipos de bases de datos se encuentran las que permiten almacenar datos semiestructurados?
Relacionales
Jerárquicas
Distribuidas
*NoSQL
EXPL:Las bases de datos NoSQL (MongoDB, Cassandra, Redis) están diseñadas para datos semiestructurados o no estructurados. Las relacionales requieren esquema fijo. Las jerárquicas tienen estructura de árbol rígida.
¿Qué tipo de bases de datos permiten incluir herencia en su diseño?
Jerárquicas
Relacionales
Distribuidas
*Orientadas a objetos
EXPL:Las bases de datos orientadas a objetos permiten herencia entre clases de objetos, igual que en la programación orientada a objetos. Las relacionales no tienen herencia nativa; las jerárquicas tienen estructura padre-hijo pero no herencia.
¿Qué lenguaje usa la inmensa mayoría de bases de datos relacionales?
C++
Java
Python
*SQL
EXPL:SQL (Structured Query Language) es el lenguaje estándar para gestionar bases de datos relacionales. Permite definir estructuras (DDL), manipular datos (DML) y controlar accesos (DCL).
¿Qué tipos de bases de datos permiten reducir los cuellos de botella y aumentar la disponibilidad de los datos?
Centralizadas
Jerárquicas
Relacionales
*Distribuidas
EXPL:Las bases de datos distribuidas reparten los datos entre múltiples nodos, lo que reduce los cuellos de botella y aumenta la disponibilidad. Si falla un nodo, los demás siguen funcionando.
¿Quién acuñó la palabra "informatique" en 1962, de la que deriva el término informática?
Philippe Doumar
Philippe Chason
Charles Babbage
*Philippe Dreyfus
EXPL:Philippe Dreyfus acuñó el término "informatique" en 1962 como contracción de "information automatique". En español se adoptó como "informática". Charles Babbage fue el inventor de la máquina analítica en el siglo XIX.
