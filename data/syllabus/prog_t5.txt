Aquí te dejo 8 preguntas tipo test avanzadas basadas en el texto proporcionado:

1. ¿Cuál es el requisito absoluto para utilizar la búsqueda binaria en un array?

* El array debe estar ordenado
* El array debe tener un tamaño fijo
* El array debe ser de tipo String

EXPL: La búsqueda binaria requiere que el array esté ordenado previamente para funcionar correctamente.

2. ¿Cuál es la eficiencia de la búsqueda lineal en un array de 1 millón de datos?
* Alta
* Media
* Baja
* Instantánea

EXPL: La búsqueda lineal tiene una eficiencia baja, ya que puede requerir hasta 1 millón de comparaciones para encontrar el dato buscado.

3. ¿Cuál es el algoritmo utilizado en la búsqueda binaria?
* Divide y Vencerás
* Recursividad
* Iteración
* Selección

EXPL: La búsqueda binaria utiliza el algoritmo "Divide y Vencerás", que consiste en dividir el array en mitades y buscar el dato en una de ellas.

4. ¿Cuál es el resultado de la búsqueda lineal si el dato buscado no se encuentra en el array?
* Se devuelve el índice del dato siguiente
* Se devuelve el índice del dato anterior
* Se devuelve -1
* Se lanza una excepción

EXPL: La búsqueda lineal devuelve -1 si el dato buscado no se encuentra en el array.

5. ¿Cuál es la ventaja de utilizar la búsqueda binaria en lugar de la búsqueda lineal?
* Es más fácil de programar
* Es más lenta
* Es más eficiente
* Es más segura

EXPL: La búsqueda binaria es más eficiente que la búsqueda lineal, ya que requiere menos comparaciones para encontrar el dato buscado.

6. ¿Cuál es el tipo de datos que se puede buscar utilizando la búsqueda binaria?
* Solo números enteros
* Solo cadenas de texto
* Cualquier tipo de dato
* Solo arrays de objetos

EXPL: La búsqueda binaria se puede utilizar para buscar cualquier tipo de dato que se pueda comparar.

* Se empieza en la primera posición y se busca hacia adelante
* Se busca en la mitad del array
* Se busca en la posición aleatoria

EXPL: La búsqueda lineal se empieza en la primera posición del array y se busca hacia adelante hasta encontrar el dato buscado.

8. ¿Cuál es la característica de la búsqueda binaria que la hace más rápida que la búsqueda lineal?
